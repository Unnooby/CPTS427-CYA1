# CPTS427-CYA1
Completion Proof of CPTS 427 - CYA 1

Objectives:

Identify your short-term and long-term career goals
Select and complete a career-aligned experience of your choice
Produce a tangible, verifiable deliverable demonstrating effort and engagement
Reflect critically on your decisions, performance, and professional growth  
Examples of Eligible Adventures (non-exhaustive):

Contribute to an open-source project (e.g., resolve a documented issue, submit a pull request, improve documentation, add a feature)
Apply for on-campus jobs, either part-time or full-time
Apply for internships or full-time roles
If already employed, take on and complete a stretch goal or demonstrate measurable impact in your current goals
Participate in a professional networking event
Feel free to email the instructor to verify if an activity not listed here could qualify as a CYA
Expected time commitment:

How many issues to complete for an open-source project or for an existing job?
How many jobs to apply?
Depends upon the intensity of the tasks
This assignment provides 6% credit towards the final grade. This is a 3-credit class. WAC 250-61-050 requires three hours expected time commitment per credit.
Result: [3 * 3 * 16] * 0.06 = 8.64 hours.
Instructions:

Produce your deliverable and reflection in a GitHub repo. Public visibility settings encouraged unless the deliverable has commercial and/or intellectual property prospects
Repo should have a README with:
A clear description of the chosen adventure
Context about your career goals
A summary of the deliverable
Deliverable-specific content, as applicable:
Code contributions, pull request links, or issue references for open source work
Application materials for job apps
Documentation of stretch goals accomplished if already working as an intern or a researcher
Screenshots, spec files, and/or other relevant assets/files/artifacts
Reflection markdown report, maximum of 500 words
Activity description: what were the expectations and what did you do?
Technical or professional decisions: what key decisions did you make (technical, strategic, or interpersonal)? Why?
Contributions: did you work independently or with others? If in a team or workplace setting, what were your specific individual contributions?
Quality assessment: How do you assess your performance? What would you improve if you were to repeat this experience?
Career alignment: How did this experience move you closer to your career goals?
